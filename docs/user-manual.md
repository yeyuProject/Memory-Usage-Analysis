# Memory Usage Analysis 用户手册

## 简介

Memory Usage Analysis 是一款Windows程序内存占用分析工具，帮助您监控和分析应用程序的内存使用情况。

## 安装

### 方式一：安装程序
1. 下载 `Memory-Usage-Analysis-Setup.exe`
2. 双击运行安装程序
3. 按照提示完成安装

### 方式二：便携版
1. 下载 `Memory-Usage-Analysis-Portable.exe`
2. 双击即可运行，无需安装

## 功能说明

### 1. 仪表盘
- 显示系统概览信息
- 快速选择要分析的进程
- 显示选中进程的内存分布饼图

### 2. 实时监控
- 实时显示进程内存使用情况
- 支持工作集、私有工作集、提交大小三个指标
- 显示内存使用趋势图
- 可配置采集频率（500ms/1秒/2秒/5秒）

**使用步骤：**
1. 在仪表盘选择要监控的进程
2. 点击"实时监控"菜单
3. 点击"开始监控"按钮
4. 观察内存使用变化

### 3. 数据录制
- 录制进程内存使用数据
- 支持自定义采样间隔和录制时长
- 可查看录制历史和详情

**使用步骤：**
1. 选择要录制的进程
2. 点击"新建录制"
3. 设置采样间隔和录制时长
4. 点击"开始录制"
5. 录制完成后可查看历史数据

### 4. 数据分析
提供三种图表类型：
- **饼图**：显示内存使用分布
- **折线图**：显示内存使用趋势
- **柱状图**：对比多个进程的内存使用

### 5. 数据筛选
- 按进程筛选
- 按内存指标筛选
- 按时间范围筛选

### 6. 通知功能
- 设置内存使用阈值
- 当内存超过阈值时自动提醒
- 查看通知历史记录

**使用步骤：**
1. 点击"通知设置"
2. 选择监控指标
3. 设置阈值（MB）
4. 点击"添加规则"

### 7. 报告导出
支持三种导出格式：
- **CSV**：适合在Excel中分析
- **HTML**：适合在浏览器中查看
- **JSON**：适合程序处理

**使用步骤：**
1. 完成数据录制或监控
2. 点击"报告导出"
3. 选择导出格式
4. 点击"导出报告"

### 8. 设置
- 切换语言（中文/英文）
- 切换主题（浅色/深色）

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+R | 刷新进程列表 |
| Ctrl+S | 保存当前数据 |
| Ctrl+E | 导出报告 |

## 常见问题

### Q: 为什么看不到某些进程？
A: 需要以管理员权限运行才能查看所有进程。

### Q: 录制数据保存在哪里？
A: 数据保存在浏览器本地存储中，清除浏览器数据会丢失录制记录。

### Q: 如何提高监控精度？
A: 使用更短的采样间隔（如500ms），但会增加CPU使用率。
